Python 3.10 Environment Setup Guide


❓ Why Specific Older Versions?

You might wonder why we target Python 3.10 and MediaPipe 0.10.14 instead of just installing the latest versions.

The newest updates to MediaPipe have started transitioning away from the classic Solutions API (which our hand-tracking code relies on) toward a new system. Using the latest versions may cause the code to break or spam the terminal with "legacy" warnings.

By locking down Python 3.10 and MediaPipe 0.10.14, we guarantee a stable environment where the gesture-control code runs flawlessly on the first try — no version conflicts.


🐍 Step 1: Download & Install Python 3.10


Go to the official website: python.org/downloads
Look for Python 3.10.x (e.g., 3.10.11) and download the installer for your OS (Windows 64-bit recommended)
Run the downloaded .exe installer


⚠️ Important: Leave the "Add python.exe to PATH" checkbox unchecked




Click Install Now and complete the installation



📁 Step 2: Open a Fresh Project Folder


Create a new folder on your computer and name it after your project (e.g., Gesture_Arcade)
Open Visual Studio Code
Go to File > Open Folder and select your new folder



🔧 Step 3: Create a Virtual Environment


Open the Command Palette with Ctrl + Shift + P
Type and select: Python: Select Interpreter
Click Create virtual environment → Venv → Select Python 3.10 (64-bit)
Name the folder .venv and hit Enter



📦 Step 4: Install Packages


Open a new terminal in VS Code
Make sure (.venv) appears at the start of the terminal line
If not, activate the environment manually:


bash   .\.venv\Scripts\Activate


If that doesn't work, look up the activation command for your specific terminal type (PowerShell, CMD, Git Bash, etc.)




Run the following install command:


bash   pip install numpy matplotlib opencv-python mediapipe==0.10.14 pycaw


✅ You're Ready to Code!

Once the installation finishes, your isolated environment is fully configured with every dependency needed to run the project — from core math logic to real-time computer vision and audio controls.

Simply drop your project code into the folder, hit Run, and it will work.
